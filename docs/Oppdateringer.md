# OBS - oppdateringer

### Ukentlig oppdateringer
På mandager kjøres det en oppdatering av Helsedatasenter, og påfølgende natt (til tirsdag) klokken 03:00 restartes hele systemet. Dette er ukentlig prosedyre. Arbeid med endring og lagring som pågår rundt klokken 03:00 natt til tirsdager vil dermed bli avbrutt. Unngå behandling av data i server ved dette tidspunktet.
