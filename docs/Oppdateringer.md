# Viktig informasjon

### Ukentlige oppdateringer
På mandager kjøres det en oppdatering av Helsedatasenter, og påfølgende natt (til tirsdag) klokken 03:00 restartes hele systemet. Dette er ukentlig prosedyre. Arbeid med endring og lagring som pågår rundt klokken 03:00 natt til tirsdager vil dermed bli avbrutt. Unngå behandling av data i analyserom ved dette tidspunktet.

### Systemvedlikehold
Vi sender ut varsel via e-post om midlertidig tjenesteavbrudd i forbindelse med teknisk vedlikehold av Analyserom. Noen ganger kan slike varsler komme kort tid før tjenesteavbruddet inntreffer.
