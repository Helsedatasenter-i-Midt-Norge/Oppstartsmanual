# LibreOffice

_For bedre funksjonalitet uten internett, og av lisensieringshensyn, er Analyserom utstyrt med LibreOffice i stedet for Microsoft Office 365._

---

## Inkluderte applikasjoner

LibreOffice inkluderer følgende:

| Applikasjon| Funkjson | Tilsvarer i Office 365 |
| -----------| ---------- | -----------|
| LibreOffice Writer | tekstbehandler |Microsoft Word|
| LibreOffice Calc | regneark |Microsoft Excel|
| LibreOffice Impress | lysbilde |Microsoft Powerpoint|
| LibreOffice Draw | vektorgrafikk |Microsoft Designer|
| LibreOffice Base | databasehåndtering |Microsoft Access|
| LibreOffice Math | formelredigering |Microsoft Equation Editor|



## Kompatibilitet med Office 365

LibreOffice benytter åpne filformater som standard, særlig OpenDocument Format (ODF), med filtyper som .odt (tekst), .ods (regneark) og .odp (presentasjoner), men det kan også åpne og lagre filer i Microsoft Office-formater som .docx, .xlsx og .pptx. LibreOffice er toveis kompatibelt med Office 365, noe som betyr at brukere kan utveksle filer mellom systemene uten betydelig tap av innhold eller funksjonalitet, selv om mindre formateringsforskjeller kan forekomme.
