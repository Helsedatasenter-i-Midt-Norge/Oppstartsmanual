# PyCharm

_Integrert utviklingsmiljø (IDE) brukt til programmering i Python_

---
## Installere PyCharm-plugins i et analyserom

### Last ned plugin på en maskin med internett

På en PC med nettilgang:
- Gå til [JetBrains Plugin Marketplace](https://plugins.jetbrains.com/)
- Finn ønsket plugin
- Åpne pluginsiden og last ned riktig versjon som `.zip` eller `.jar`
-  Viktig å sjekke:
   - At pluginen er kompatibel med installert PyCharm-versjon
   - Eventuelle avhengigheter (dependencies)
- Overfør filen til analyserom via Filemail eller FileSender


### Installer plugin fra disk i PyCharm

- Åpne Pycharm i analyserommet
- Gå til File → Settings → Plugins
- Klikk på ⚙️ (tannhjul‑ikonet)
- Velg "Install Plugin from Disk…"
- Velg `.zip` eller `.jar`-filen du overført
- Start PyCharm på nytt hvis du blir bedt om det
